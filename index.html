Redirecionando para o PDF...

Neste exemplo, a página HTML contém uma meta tag que realiza um redirecionamento automático para o arquivo PDF após um atraso de 0 segundos. Os visitantes verão a mensagem "Redirecionando para o PDF..." antes de serem redirecionados para o PDF. Lembre-se de substituir "seu-arquivo.pdf" pelo caminho correto para o arquivo PDF que você deseja que seja acessado quando alguém entrar no site. No entanto, é importante considerar que forçar um PDF como página inicial pode não proporcionar uma experiência ideal para os usuários, especialmente aqueles em dispositivos móveis. Muitos navegadores e dispositivos têm mecanismos de visualização de PDF embutidos, mas eles podem não ser tão convenientes quanto a visualização de uma página HTML. Portanto, certifique-se de que essa abordagem atenda às necessidades dos seus usuários.
